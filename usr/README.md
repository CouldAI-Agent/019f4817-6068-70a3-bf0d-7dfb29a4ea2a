# Ігровий Асистент (Gaming Assistant)

Повноцінний кросплатформений додаток, створений на Flutter, який виступає в ролі вашого ідеального ігрового компаньйона. Він містить інструменти для керування ботами, базу даних чіт-кодів для різних ігор та спеціальний розділ з інформацією по всесвіту Dota.

## Можливості
- **Керування Ботами**: Інтерфейс для моніторингу та управління автоматизованими скриптами.
- **База Чіт-кодів**: Легкий доступ до популярних чіт-кодів з можливістю їх копіювання.
- **Dota Інфо**: Перегляд інформації про героїв, гайди та статистики.
- **Адаптивний Дизайн**: Ідеально працює на мобільних пристроях, планшетах та десктопах завдяки підтримці `NavigationRail` та `NavigationBar`.
- **Темна Тема**: Створений спеціально для геймерів із використанням темної кольорової схеми (Material 3).

## Технологічний стек
- **Flutter**: Фреймворк для кросплатформеної розробки.
- **Dart**: Мова програмування.

## Запуск проекту
1. Встановіть [Flutter SDK](https://docs.flutter.dev/get-started/install).
2. Клонуйте цей репозиторій.
3. Виконайте команду `flutter pub get` для встановлення залежностей.
4. Запустіть додаток командою `flutter run`.
